Fake News Classifier

CountVectorizer Approach

1. Taking data from Kaggle with 20800 data points

2. Preprocessing: Remove special characters, stop words, and convert to lowercase

3. Use bag of words method with 5000 max features and 3-word n-grams

4. Train-test split with 33% test size

5. Train seven different ML algorithms

TF-IDF Approach

1. Use train (20800 data) and test (5200 data) from Kaggle

2. Preprocessing: Create features from News Title, Whole News, and News Author

3. Use TF-IDF transformer for feature matrix

4. Default train-test split

5. Train six different ML algorithms

LSTM Approach

A sequential deep learning model using LSTM architecture for binary text classification

Dataset size: 20800

Pretrained GloVe text embedding used for vectorization

Achieved around 99% accuracy
